# Scoring Rules

Position A: Integration Developer - Oracle EBS

- Technical Match: 40%
- Oracle EBS Match: 25%
- Experience Match: 15%
- Language Match: 10%
- Industry Match: 10%

Position B: Business Systems Analyst - Oracle EBS

- Functional Match: 35%

- Oracle EBS Match: 20%
- Leadership Match: 10%
- Language Match: 10%

Classification:

- scoreA >= 80 and scoreB < 80: Position A
- scoreB >= 80 and scoreA < 80: Position B
- scoreA >= 80 and scoreB >= 80: Both
- Otherwise: No Match
